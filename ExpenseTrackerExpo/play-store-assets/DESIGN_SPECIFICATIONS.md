## ğŸ“‹ Step-by-Step Creation Guide

### Using Canva (Easiest):

1. **Go to Canva.com** and sign up (free)
2. **Create Custom Size:** 1024 x 500 pixels
3. **Add Background:**
   - Use gradient: Purple (#667eea) to Blue (#007AFF)
   - Or solid color with pattern
4. **Add App Icon:**
   - Upload your app icon or use emoji
   - Size: 120x120px
   - Position: Center-top
5. **Add App Name:**
   - Font: Bold, Sans-serif
   - Size: 48px
   - Color: White
   - Position: Below icon
6. **Add Tagline:**
   - Font: Regular, Sans-serif
   - Size: 24px
   - Color: White (slightly transparent)
   - Position: Below app name
7. **Add Feature Icons:**
   - Use emoji or icon library
   - Add text labels below
   - Arrange horizontally
8. **Add Decorative Elements:**
   - Circles, lines, or shapes
   - Use low opacity
9. **Export:**
   - Download as PNG
   - Ensure dimensions are 1024 x 500

### Using Figma (Professional):

1. **Create New File**
2. **Set Frame Size:** 1024 x 500
3. **Add Rectangle:** Full size, apply gradient
4. **Add Elements:** Icon, text, features
5. **Export:** PNG, 2x resolution (2048 x 1000 for retina)
